The Spreadsheet Whisperer: Prompts That Actually Understand Your Data

Source ↗ Prompting

What This Skill Is

As of 2026 you have two genuinely powerful paths for AI spreadsheet work: Claude for Excel (better for multi-tab workbook analysis and formula reasoning) and ChatGPT for Excel (better for financial modeling and live market data). This skill covers the 4-step analysis framework and the prompts that actually get deep results from your data.

Why It Works

When you upload .xlsx to ChatGPT directly, it captures static values and loses all formula logic. Claude for Excel reads live formulas, provides clickable cell references, makes direct edits while preserving dependencies, and navigates across dozens of tabs. The "before/after formula" rule in the prompt means you can audit every change visually before it's applied. The "Questions I should ask" section forces the model to surface what the data implies but doesn't answer.

Security warning: Both add-ins warn that prompt injection attacks can hide malicious instructions inside spreadsheet cells. Only use with trusted spreadsheets — never with downloaded vendor templates from the internet.

How to Use It

Setup for Claude for Excel (2 minutes):

  1. Open Excel → Home → Add-ins → search "Claude by Anthropic" → install
  2. Press Ctrl+Alt+C (Windows) or Ctrl+Option+C (Mac) to toggle the Claude sidebar
  3. Sign in with your paid Claude account

The 4-step analysis framework:

  1. Explore: "Tell me what's in this workbook. Which tabs relate to each other? Flag anything unusual."
  2. Diagnose: "Why did [metric] change? Show me the formula chain that drives it."
  3. Predict: "If [variable] changes by X%, show me the downstream effect across all tabs."
  4. Prescribe: "Build a summary sheet with the three recommendations a finance director would make from this data."

The audit trick: If a financial assumption doesn't update the right downstream cell, highlight it and ask "Why isn't this cell linked to [x]?" The value of AI in Excel isn't in the build — it's in the audit.

Claude vs. ChatGPT for Excel:

  • Claude: stronger at multi-tab complex workbooks, explaining formula chain behavior
  • ChatGPT: stronger at live financial data from FactSet/S&P/LSEG, building models from scratch

Prompt Template

I have [describe your dataset: rows, columns, what each tab contains].

Step 1: Create a dashboard in a new sheet called "Executive Summary" with key trends.
- Use Python scripts for any calculation Excel formulas can't handle cleanly
- Include 3-5 KPI cards at the top (use conditional formatting for up/down arrows)
- Add 2-3 charts: one time series, one comparison, one distribution
- Add a "What changed vs. last period" callout box

Step 2: Format static input cells in BLUE and formula-linked cells in BLACK.
This lets me audit your work visually.

Step 3: Add a "Questions I should ask" section at the bottom with 3 analytical
questions this data surfaces but doesn't answer.

Before making any change to existing cells, show me the before/after formula.
Don't overwrite anything without approval.
