Stop Saving Prompts in Docs — Use Claude Skills Instead

Source ↗ Prompting

What This Skill Is

If you've been copy-pasting your best prompts from a Google Doc every time you need them, Claude Skills is the upgrade. Reusable instruction sets you install once and trigger with a single slash command — in any chat.

Alex Pereira built six of them to run his entire content workflow (hook generator, image creator, video generator, thumbnail builder, face lock, caption writer) — all fired with one word.

Alex's rule for when to build a skill: the task must be recurring, structured, and have a consistent output format. All three? Build it. One or two? Keep it as a saved prompt.

Why It Works

The friction of finding and copy-pasting a saved prompt is enough to make you not use it. A slash command removes that friction entirely — the skill is always one word away, in any context.

How to Use It

  1. In Claude, click Customize → + → Create Skill
  2. Structure it in three parts: (1) description + when Claude should trigger it, (2) instructions using the template above, (3) examples of good output
  3. Save it — then type /[skill name] in any chat to activate
  4. Start with your most repeated prompt (the one you've copy-pasted more than 10 times)

Prompt Template

When I type /[skill name], do the following:

Instructions: [What you want Claude to do]
Format: [Output shape — e.g. "10 hooks, numbered list, under 15 words each"]
Voice rules: [Any style/tone requirements]
Examples of good output:
1. [Example 1]
2. [Example 2]
