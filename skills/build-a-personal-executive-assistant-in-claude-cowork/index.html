Build a Personal Executive Assistant in Claude Cowork (Including Web Scraping, No Zapier Required)

Source ↗ Productivity

What This Skill Is

As of Q1 2026, you can build a full personal executive assistant in Claude Cowork with web scraping, email management, calendar coordination, and scheduled briefings — entirely natively, no Zapier or other middleware required. This is a 7-step complete setup guide.

Why It Works

The dedicated assistant email is the single most important setup step most people skip. It isolates AI-generated correspondence from your main identity and gives you a natural audit trail. Explicit hard policies (NEVER send, NEVER delete) are what separate a useful system from a dangerous one. Once your policies are in place, the system operates like a real executive assistant who never forgets context and never needs to be brought up to speed.

How to Use It

The 7-step setup:

  1. Download Claude Desktop (macOS or Windows) from claude.ai/download
  2. Create a dedicated assistant email (e.g., yourname.assistant@gmail.com) with delegated access to your calendar — keeps AI-generated correspondence isolated
  3. Connect Gmail and Calendar natively: Customize → Connectors (no third-party middleware needed)
  4. Install Firecrawl for web scraping: In Claude Desktop, run /plugin and search for Firecrawl — gives Claude the ability to scrape any public website including JavaScript-heavy pages (500 free credits/month)
  5. Write hard policies, not guidelines: Create assistant-policies.md with explicit rules for email drafting, calendar changes, research citations, and off-limits scope
  6. Schedule your morning briefing: In Cowork, type "schedule" followed by your task description (Mac/PC must stay awake with Claude Desktop open)
  7. Save workflows as Skills: After iterating a workflow into something reliable, ask Claude to use its skill creator to package it

Key policy file to add to your project knowledge:

# Email Rules
- NEVER send emails directly. Always draft and surface for my approval.
- VIP senders (always flag): [list]

# Calendar Rules
- NEVER delete calendar events. Propose a change; I approve.
- NEVER accept meeting invites on my behalf. Flag and wait.

# Scope Boundaries
- OFF-LIMITS: financial transactions, medical decisions, legal communication

Prompt Template

Every weekday at 7am, run my morning briefing:

1. Pull all emails received since 6pm yesterday. Categorize into:
   URGENT (needs reply today), IMPORTANT (this week), FYI (archive later).
   For URGENT, draft a reply I can send in under 2 minutes each.

2. Pull my calendar for today. For each meeting:
   - Name the attendees
   - Pull the last email thread with each attendee
   - Flag any I haven't prepared for
   - Draft 3 talking points per meeting

3. Use Firecrawl to check for news mentions of:
   - My company: [company name]
   - Our top 5 competitors: [list]
   - My 3 priority accounts: [list]
   Only surface items from the last 24 hours.

4. Output as a single email draft to my assistant address with subject
   "Morning briefing [date]". Do not send.

Checkpoint in on any task that takes longer than 5 minutes.
