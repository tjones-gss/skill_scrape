How to Prompt Claude Fable 5 From the Leaked System Prompt

Source ↗ June 10, 2026 Prompting

What This Skill Is

Prompt patterns derived from the publicly circulated Claude Fable 5 system prompt (treat as a third-party artifact, not an official source). The key insight: Fable's best users prompt it like an operating system for work, not like a chatbot.

Why It Works

The system prompt reveals how Claude is trained to behave: it wants current docs, structured prompts, explicit constraints, source priority, and clear formats. Prompting against those defaults wastes its capabilities. Prompting with them unlocks the model's full ability to run for hours on complex, long-horizon work.

How to Use It

Five patterns derived from the system prompt:

  1. For product questions: "Check Anthropic docs and support first, then explain the current behavior." Claude's product knowledge may be stale.
  2. For high-stakes work: Include audience, format, scope, sources, success criteria, allowed tools, forbidden moves, and review requirements upfront.
  3. For ambiguous requests: Give constraints before asking — don't rely on the model to ask clarifying questions mid-run.
  4. For scannable output: Explicitly ask for headings and bullets; default is prose.
  5. For research with current info: Specify source priority — "Use Anthropic docs first, then primary sources, then high-quality secondary coverage."

Prompt Template

Task: [describe the work]

Before starting:
- Check [specific docs/source] first for any product, policy, or version details
- Use this source priority: [internal docs > primary sources > secondary coverage]

Constraints:
- Audience: [who will read or use this]
- Format: [prose / bullets / table / code]
- Scope: [what's in and out of scope]
- Success criteria: [what a good result looks like]
- Do not: [forbidden moves, things to avoid]
- Review requirement: [what I'll check before using the output]

Work step by step. If you are uncertain about any fact, flag it rather than guessing.
Return the output in the format specified. Do not over-format if prose was requested.
